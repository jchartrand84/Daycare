Daycare Management System
=========================

Description:
------------
This project is a Daycare Management System developed in Python using the Tkinter library for the graphical
user interface (GUI). It provides a comprehensive solution for managing a daycare center, including features for
managing a database of children, processing payments, and managing a calendar for scheduling.

Features:
---------
- Child Management: Add, update, and delete child information in the database.
- Payment Processing: Handle payment transactions and maintain records.
- Calendar Management: Schedule and manage events and activities in the daycare center.

Requirements:
-------------
- Python 3.12.2
- Tkinter library (usually included with Python)
- tkcalendar library (install using `pip install tkcalendar`)

Installation:
-------------
1. Ensure that Python 3.12.2 is installed on your system.
2. Clone or download the project repository to your local machine.
3. Run the main Python file to launch the application.

Usage:
------
1. Launch the application.
2. Use the GUI to navigate through the different functionalities of the system.
3. Add, update, or delete child information as needed.
4. Process payments and manage the calendar through the respective modules.
